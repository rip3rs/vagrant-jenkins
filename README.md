# Vagrant-Jenkins.
Vagrant box with Nginx Jenkins.

### Prerequisities

`Vagrant`
`virtualbox`

### Instalation

$`vagrant up`

$`sudo vim /etc/hosts`

paste
```
192.168.33.99   jenkins.local.dev
```

It should go straight to the working folder inside of the box
`/var/www/jenkins.local.dev`

`http://jenkins.local.dev`
